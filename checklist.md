## Phase 2. 신호 생성 모듈

- [x] MA20 / MA60 추세 필터 구현 (`kis_indicator.py`)
- [x] 20일 모멘텀 스코어 계산 (`kis_momentum.py`)
- [x] 섹터 분류 및 주도 섹터 판별 (`kis_sector.py`)
- [x] 외국인 5일 누적 순매수 필터 (`kis_foreign.py`)
- [x] 3단계 필터 AND 결합 → 최종 매수 신호 생성
  - 1단계: 섹터 모멘텀 상위 3개 섹터 선정
  - 2단계: 섹터 내 상승추세 (현재가 > MA20 > MA60)
  - 3단계: 외국인 5일 순매수 > 0
  - 섹터당 대장주 1종목만 선정 (최대 3종목)
