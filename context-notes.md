# 한국 주식 자동매매 시스템 — 컨텍스트 노트

## 프로젝트 개요

코스피 대형주 위주로 외국인 프로그램 매매 동향을 참고하여 여러 종목을 자동으로 매매하는 시스템을 구축한다.

---

## 핵심 전략

### 기반 전략: 외국인 프로그램 매매 추종

외국인 프로그램 매매의 대부분은 **비차익거래** — 글로벌 펀드가 포트폴리오 내 한국 비중을 조절하기 위해 코스피 200 시가총액 상위 종목을 지수 비중에 맞게 일괄 매수/매도하는 방식.

- 프로그램 순매수 지속 = 글로벌 자금 한국 유입 신호
- 프로그램 순매도 지속 = 한국 비중 축소 신호

### 추가 전략 1: 추세 필터 (이동평균)

- MA20 / MA60 상향돌파 상태인 종목만 진입 허용
- 프로그램 매수 신호가 와도 하락 추세 종목은 제외
- 두 조건 모두 충족 시 진입 (MA20 > MA60, 현재가 > MA20)

### 추가 전략 2: 신호 강도 임계값

- 프로그램 순매수 금액이 일정 임계값(N억) 이상일 때만 유효 신호로 처리
- 소규모 프로그램 매매는 노이즈로 간주하여 무시
- 임계값은 백테스트 후 결정 (미결정)

### 추가 전략 3: 모멘텀 스코어링 (종목 선정)

- 코스피200 중 최근 20일 수익률 상위 N종목만 편입 대상으로 선정
- 외국인 순매수 + 모멘텀 상위 = 진입 확신 강화
- 편입 종목 수 N은 미결정

---

## 매매 신호 해석 매트릭스

| 신호 조합 | 해석 |
|-----------|------|
| 프로그램 매수 + 논프로그램 매수 | 강한 유입. 대형주 전반 상승 |
| 프로그램 매수 + 논프로그램 매도 | 시장은 들어오나 특정 섹터 차익실현 중 |
| 프로그램 매도 + 논프로그램 매수 | 시장 비중은 줄지만 특정 종목 집중 배팅 |
| 프로그램 매도 + 논프로그램 매도 | 강한 유출. 매수 보류 |

---

## 주의 이벤트

- 선물 만기일: 매월 2번째 목요일 — 장 막판 프로그램 청산으로 지수 급등락 가능.
- 쿼드러플 위칭(분기별): 선물+옵션 동시 만기 — 변동성 극대화 구간.
- 사이드카: 선물 ±5% 1분 지속 시 발동 → 프로그램 매매 5분 정지.
- 서킷브레이커: 코스피 -8% 1분 지속 시 발동 → 전체 매매 20분 중단.

## 확정 항목

| 요소 | 결정값 |
|------|--------|
| 운용 규모 | 1,000만원 |
| 손절 기준 | 진입가 대비 -10% |
| 익절 기준 | 트레일링 스탑 — 고점 대비 -10% 하락 시 청산 |
| API | 한국투자증권 KIS API |
| 언어 | Python |

## 미결정 항목

| 요소 | 역할 |
|------|------|
| 프로그램 순매수 금액 임계값 | 신호 강도 필터 |
| 연속 N일 순매수 조건 | 추세 확인 |
| 편입 종목 수 | 모멘텀 상위 N종목 |
| 매매 주기 (일봉/단기) | 전략 설계 기반 |
| 실행 환경 | 개인 PC / 클라우드 |

---

## 기술 스택 (확정)

- API: 한국투자증권 KIS API (REST + WebSocket, 공식 Python SDK)
- 실행 환경: 미선택 (개인 PC Windows / 클라우드)
- 언어: Python
- 데이터 소스: KRX 실시간 프로그램 매매 공시 활용 예정
